# @for-yeyu/evm-dapp-skills

Standard skills package for the EVM DApp layered architecture.

## Install

```bash
npx skills add "https://github.com/for-yeyu/evm-dapp-starter-for-yeyu/tree/main/.agents/skills" -y
```

After installation, skills are automatically synced to:

- `<your project>/.agents/skills` 

## Included Skills

- `evm-dapp-standard-coding`
- `app-router-conventions`
- `ui-conventions`
- `api-conventions`
- `hooks-conventions`
- `configs-conventions`
- `lib-infrastructure-conventions`
- `styles-conventions`
